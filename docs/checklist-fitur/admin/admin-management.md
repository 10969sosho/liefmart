# Checklist Fitur: Admin Management

Pengaturan hak akses dan pengguna sistem.

**File View**: `resources/views/admin/roles/index.blade.php`, `admin/users/index.blade.php`

## 1. Role Management (Peran)
- [ ] **List Role**: Super Admin, Gudang, Admin Sales, Finance.
- [ ] **Permission Matrix**: Checklist hak akses (View, Create, Edit, Delete) per modul.

## 2. User Management (Pengguna)
- [ ] **List User**: Nama, Email, Role.
- [ ] **Reset Password**: Admin bisa reset password user lain.
- [ ] **Assign Role**: Ubah peran user.
- [ ] **Status**: Aktif / Suspend user.
